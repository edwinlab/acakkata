# Game Acak kata
[![Build Status](https://api.travis-ci.com/edwinlab/acakkata.svg?branch=master)](https://travis-ci.com/edwinlab/acakkata)

## Installation

execute:

```
$ bundle
```

## Usage

```
$ chmod +x bin/run
$ ./bin/run
```

## Run Test
```
$ rspec spec
```
